# Vanitas History

Vanitas 프로젝트의 날짜별 진행 기록입니다.

## 기록 규칙

- 작업이 끝날 때마다 해당 날짜 아래에 `질문`, `답변/결정`, `작업`, `검증`, `업로드`를 간단히 남긴다.
- API 키, DB 접속 문자열, 비밀키, 개인 계정 정보는 기록하지 않고 `[비공개]`로 적는다.
- GitHub/Vercel에 올릴 변경이 있으면 로그를 먼저 갱신한 뒤 함께 커밋한다.
- 사용자의 원문 질문은 필요한 의미만 요약하고, 긴 대화는 결정 사항 중심으로 압축한다.
